# 🚧 Smart Road Safety System

A **community-driven road safety platform** that allows users to report road hazards in real-time and receive intelligent alerts to avoid accidents.

---

## 🌟 Features

* 🛣️ **Real-Time Hazard Reporting**
  Report potholes, accidents, waterlogging, and road issues instantly.

* 🔔 **Smart Alerts System**
  Get notified about hazards near your location.

* 🤖 **AI-Based Analysis**
  Identify high-risk zones and predict accident-prone areas.

* 👥 **Community Verification**
  Users can validate reports for better accuracy.

* 📊 **Analytics Dashboard**
  Visualize road safety data and trends.

* 🎮 **Driving Simulator**
  Practice safe driving in a simulated environment.

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Sachin976347/Smart_Road_Safety.git
cd Smart_Road_Safety
```

---

### 2️⃣ Setup Virtual Environment

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
```

---

### 3️⃣ Install Dependencies

```bash
pip install flask flask-cors flask-jwt-extended
```

---

### 4️⃣ Run Backend Server

```bash
python app.py
```

Server will start at:

```
http://localhost:5000
```

---

### 5️⃣ Run Frontend

* Open `index.html` in browser
* OR use Live Server (VS Code recommended)
